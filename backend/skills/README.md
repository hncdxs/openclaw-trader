# 🦞 OpenClaw Workspace Skills

skills/ 目录包含交易策略技能。每个技能是一个子目录，包含 SKILL.md。

结构：
```
skills/
├── crypto-trader/          # 基础交易员技能（必选）
│   └── SKILL.md
├── btc-scalping/           # BTC 短线策略示例（可自定义）
│   └── SKILL.md
└── trend-following/        # 趋势跟踪策略示例（可自定义）
    └── SKILL.md
```

技能通过 Web UI 中的"策略管理"页面进行创建/编辑/启用/禁用。
